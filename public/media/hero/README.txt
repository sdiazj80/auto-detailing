HERO MEDIA
----------
Drop the following files here to replace the cinematic gradient fallback:

  hero.mp4          1080p or 4K looping background video (muted, ~10-20s loop)
                    Suggested: slow dolly past a glossy vehicle, paint reflection,
                    foam cannon wash, or mobile rig at work.

  hero-poster.jpg   Static frame used as poster while video loads, and as
                    fallback on devices that block autoplay.

The site will automatically use these files — no code change required.
If they're missing, the hero shows a clean graphite/ice gradient instead.
